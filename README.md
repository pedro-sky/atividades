
# Atividades Preparatórias para Testes Técnicos em Processos Seletivos de Desenvolvimento de Software


## Testes Práticos para Vagas Júnior em Desenvolvimento de Software

Este repositório reúne **20 exercícios práticos de lógica de programação resolvidos**, inspirados em desafios reais aplicados em **testes técnicos para vagas de desenvolvedor(a) júnior**.

O objetivo é desenvolver raciocínio lógico, clareza de código e a capacidade de criar soluções simples, corretas e funcionais utilizando **JAVA**.

Essas são as minhas resoluções pessoais, o link para o repositório original com a resolução do autor se encontra abaixo 
https://github.com/conteudoGeneration/Atividades-preparatorias-para-teste-tecnicos-de-processos-seletivos-em-desenvolvimento-de-software/tree/main

---

## 🧪 Exercícios

### 1️⃣ Cifra de César Simples
Dada uma String e um número inteiro `n`, retorne uma nova String com as letras deslocadas `n` posições.

| Entrada | Saída |
|-------|-------|
| texto = "abc", deslocamento = 3 | "def" |

---

### 2️⃣ Par com Menor Diferença
Dado um array de inteiros e um valor alvo `x`, encontre dois números cuja soma seja `x`.

| Entrada | Saída |
|-------|-------|
| [1, 4, 7, 10], x = 11 | (4, 7) |

---

### 3️⃣ Palavra Mais Frequente
Retorne a palavra que mais aparece em uma frase (ignore maiúsculas/minúsculas).

| Entrada | Saída |
|-------|-------|
| "Java é java" | "java" |

---

### 4️⃣ Número Palíndromo
Verifique se um número é palíndromo.

| Entrada | Saída |
|-------|-------|
| 121 | true |
| 1234 | false |

---

### 5️⃣ Inverter Palavras da Frase

---

### 6️⃣ Contar Palavras na Frase

### 7️⃣ Segundo Maior Número

### 8️⃣ Verificar Anagramas
